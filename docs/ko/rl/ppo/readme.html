#

근접 정책 최적화 - PPO

이 문서는 근접 정책 최적화 - PPO의 PyTorch 구현입니다.

PPO는 강화 학습을 위한 정책 그라디언트 방법입니다. 간단한 정책 그라디언트 방법은 샘플(또는 샘플 세트)당 단일 그라디언트 업데이트를 수행합니다. 하나의 샘플에 대해 여러 단계의 그라디언트 단계를 수행하면 정책의 편차가 너무 커서 잘못된 정책이 생성되기 때문에 문제가 발생합니다. PPO를 사용하면 데이터를 샘플링하는 데 사용된 정책에 가깝게 정책을 유지함으로써 샘플당 여러 번의 그라디언트 업데이트를 수행할 수 있습니다. 업데이트된 정책이 데이터 샘플링에 사용된 정책에 가깝지 않은 경우 그라디언트 흐름을 클리핑하여 이를 수행합니다.

여기서 사용하는 실험을 찾을 수 있습니다. 실험은 일반화된 이점 추정 을 사용합니다.
