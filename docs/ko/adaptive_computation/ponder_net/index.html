#

PonderNet: Ponder 익히기 (Learning to Ponder)

논문 PonderNet: Learning to Ponder에서 소개된 PyTorch를 이용한 Ponder의 구현에 대해 설명합니다.

PonderNet은 입력에 따라 계산을 조정합니다. 순환 네트워크에서 입력에 따라 수행할 단계의 수를 변경합니다. PonderNet은 끝까지 경사 하강법을 사용하여 이를 학습합니다.

PonderNet의 단계 함수는 다음과 같은 형식을 가집니다:

y^​n​,hn+1​,λn​=s(x,hn​)

여기서 x는 입력이며, hn​ 은 상태이고, y^​n​ 은 n단계에서의 예측이며, , 그리고 λn​ 은 현재 단계에서 중단(정지)할 확률입니다.

s는 어떤 신경망이라도 될 수 있습니다. (예: LSTM, MLP, GRU, Attention layer).

n단계에서 중단할 비조건부 확률은 다음과 같습니다,

pn​=λn​j=1∏n−1​(1−λj​)

이것은 이전 단계에서 중단되지 않았고 n번째 단계에서 중단할 확률을 의미합니다.

추론 중에는 중단할 확률 λn​에 기반하여 샘플링을 하여 중단된 단계에서의 예측 y^​n​을 최종 출력으로 얻습니다.

훈련 중에는 모든 레이어에서 예측을 얻고 각각에 대한 손실을 계산합니다. 그런 다음 각 레이어에어 중단될 확률 pn​에 기반하여 손실의 가중 평균을 취합니다.

단계 함수는 N으로 표시된 최대 단계 수에 적용됩니다.

PonderNet의 전체 손실은 다음과 같습니다.

LLRec​LReg​​=LRec​+βLReg​=n=1∑N​pn​L(y,y^​n​)=KL(pn​∥pG​(λp​))​

L 은 목표 y 와 예측 y^​n​사이의 일반적인 손실 함수입니다.

KL은 Kullback–Leibler 발산입니다.

pG​는 λp​에 의해 매개변수화된 Geometric distribution(기하 분포)입니다. λp​는 λn​과 관련이 없지만, 논문의 표기법을 따르기 위해 같은 표기를 사용합니다. PrpG​(λp​)​(X=k)=(1−λp​)kλp​.

규제 손실(regularization loss)은 네트워크가 λp​1​ 단계를 취하도록 편향시키고 모든 단계에 대해 0이 아닌 확률을 의도함으로써 promotes exploration와 같이 탐색을 촉진합니다.

Parity Task를 위한 GRU를 사용한 PonderNet

이 sample은 GRU Cell을 단계 함수로 사용하는 간단한 모델입니다.

이 모델은 Parity Task 을 위한 것으로, 입력은 n_elems 의 벡터입니다. 벡터의 각 요소는 0 , 1 또는 -1 이며, 출력은 홀수 개의 1 이 있으면 참이고 그렇지 않으면 거짓인 이진값 패리티 입니다.

모델의 예측은 패리티가 1일 확률의 로그 확률입니다.

λn​=sλ​(hn​) 

146            else:

170            h = self.gru(x, h)
#

모든 샘플이 중단되면 계산을 중단합니다. 

173            if self.is_halt and halted.sum() == batch_size:

177        return torch.stack(p), torch.stack(y), p_m, y_m

재구성 손실 (Reconstruction loss)

LRec​=n=1∑N​pn​L(y,y^​n​)

L은 목표 y 와 예측 y^​n​사이의 일반적인 손실 함수입니다.

규제 손실 (Regularization loss)

LReg​=KL(pn​∥pG​(λp​))

KL은 Kullback–Leibler 발산(divergence) 입니다.

pG​ 는 λp​에 의해 매개변수화된 기하 분포(Geometric distribution)입니다. λp​ 는 λn​과 관련이 없으며, 우리는 논문의 표기법을 따르기 위해 정의했습니다. PrpG​(λp​)​(X=k)=(1−λp​)kλp​.

규제 손실은 네트워크가 λp​1​ 단계를 취하도록 편향시기고 모든 단계에 대해 0이 아닌 확률을 유도함으로써 탐색을 촉진합니다.

216class RegularizationLoss(Module):
#
  • lambda_p 는 λp​ 입니다. - 기하 분포의 성공 확률입니다.

232    def __init__(self, lambda_p: float, max_steps: int = 1_000):

237        super().__init__()

pG​(λp​)를 계산하기 위한 비어있는 벡터입니다. 

240        p_g = torch.zeros((max_steps,))
