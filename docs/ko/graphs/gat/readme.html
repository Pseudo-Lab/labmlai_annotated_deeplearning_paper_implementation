#

Graph Attention Networks (GAT)

이것은 논문 Graph Attention Networks의 파이토치 구현체입니다.

GATs는 그래프 데이터에 대해 동작합니다. 그래프는 노드와 노드를 연결하는 엣지로 구성되어 있습니다. 예를 들어, Cora 데이터셋에는 노드가 연구 논문들이고 엣지가 논문들을 연결하는 인용입니다.

GAT는 트랜스포머와 비슷하게 masked self-attention을 사용합니다. GAT는 graph attention 레이어가 서로 쌓여 구성되어있습니다. 각각의 graph attention 레이어는 입력으로 노드 임베딩값을 받고 변환된 임베딩값을 내보냅니다. 노드 임베딩값은 연결되어 있는 서로 다른 노드의 임베딩값에 집중합니다. graph attention 레이어의 자세한 사항은 구현체에 포함되어 있습니다.

여기에 Cora 데이터셋에 대하여 두 개의 GAT 레이어를 학습하는 학습코드가 있습니다.
