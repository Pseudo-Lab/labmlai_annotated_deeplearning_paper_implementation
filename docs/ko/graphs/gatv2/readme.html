#

그래프 어텐션 네트워크 v2 (GATv2)

다음은 How Attentive are Graph Attention Networks?논문의 GATv2 오퍼레이터를 파이토치로 구현한 것입니다.

GATv2는 그래프 데이터를 다룹니다. 그래프는 노드와 노드들을 연결하는 엣지로 구성되어 있습니다. 예를 들어, Cora 데이터셋에서는 노드는 연구 논문에 대응하고 엣지는 논문과 연계된 인용횟수에 대응합니다.

GATv2 오퍼레이터는 기존 GAT의 정적 어텐션 문제를 해결하였습니다: 기존 GAT의 선형 레이어들이 연속해서 적용되기 때문에, 어텐션을 계산할 때 이용되는 노드들의 랭킹이 쿼리 노드와 무관합니다. 반면, GATv2에서는 모든 노드가 다른 노드에 영향을 줄 수 있는 구조입니다.
