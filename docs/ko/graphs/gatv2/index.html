#

그래프 어텐션 네트워크 v2 (GATv2)

다음은 How Attentive are Graph Attention Networks? 논문의 GATv2 오퍼레이터를 파이토치로 구현한 것입니다.

GATv2는 GAT와 유사하게 그래프 데이터를 다룹니다. 그래프는 노드와 노드들을 연결하는 엣지로 구성되어 있습니다. 예를 들어, Cora 데이터셋에서 노드는 연구 논문에 대응하고 엣지는 논문과 연계된 인용횟수에 대응합니다.

GATv2 오퍼레이터는 기존 GAT의 정적 어텐션 문제를 해결하였습니다. 정적 어텐션이란 모든 쿼리 노드에 대해 키 노드의 어텐션이 같은 랭크(오더)를 가지는 것을 말합니다. GAT에서는 쿼리 노드 i에 대한 키 노드 j의 어텐션을 다음과 같이 계산합니다:

eij​​=LeakyReLU(a⊤[Whi​​∥Whj​​])=LeakyReLU(a1⊤​Whi​​+a2⊤​Whj​​)​

임의의 쿼리 노드 i에 대해, 키들의 어텐션 랭크 (argsort)는 a2⊤​Whj​​에 의해서만 결정됩니다. 따라서, 키들의 어텐션 랭크는 모든 쿼리에 대해 동일한(정적인) 값을 갖습니다.

GATv2는 다음과 같이 어텐션 메커니즘을 변형하여 동적 어텐션을 수행하는 구조입니다.

eij​​=a⊤LeakyReLU(W[hi​​∥hj​​])=a⊤LeakyReLU(Wl​hi​​+Wr​hj​​)​

이 논문에서는 합성 사전 룩업(synthetic dictionary lookup) 데이터셋을 사용하여, 일부 그래프 문제에 대해서는 GAT의 정적 어텐션으로 인해 학습이 실패할 수 있음을 보여줍니다. 완전 연결 이분 그래프는 한 세트의 노드(쿼리 노드)는 해당 노드들에 대응하는 키 노드들을 가지고 있고, 이 외의 노드들은 키와 키에 대응하는 벨류들을 가지고 있는 그래프입니다. 해당 문제의 목표는 쿼리 노드들의 값들을 예측하는 것입니다. GAT는 제한된 정적 어텐션으로 인해 문제를 해결하지 못하였습니다.

다음은 Cora 데이터셋에 대하여 두 개의 레이어를 가지는 GATv2를 훈련하는 훈련 코드 입니다.
