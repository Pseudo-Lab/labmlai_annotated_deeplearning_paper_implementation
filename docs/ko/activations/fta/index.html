#

퍼지 타일링 활성화 (FTA)

다음은 Fuzzy Tiling Activations: A Simple Approach to Learning Sparse Representations Online 논문의 PyTorch 구현/튜토리얼 입니다.

퍼지 타일링 활성화는 구간화(binning)를 기반으로 한 희소 활성화의 한 형태입니다.

구간화(Binning)은 주어진 구간에 따라 스칼라 값들을 구분하여 하나의 범주로 분류하는 것입니다. 구간화의 하나의 문제는 대부분의 값에 대해 (범위의 경계는 제외) 0의 기울기를 갖는다는 것입니다. 또다른 문제는 구간화는 구간의 간격이 큰 경우 정밀도를 잃을 수 있다는 점입니다.

FTA는 이러한 단점들을 보완합니다. 타일링 활성화에서와 같은 강경한 경계 대신에 각 구간 사이에 부드러운 경계를 사용합니다. 이는 모든 혹은 넓은 범위의 값들에 대해 기울기가 0이 아니게 해줍니다. 그리고 부분값에 포함되어 있어서 정밀도 또한 잃지 않습니다.

타일링 활성화

[l,u] 가 입력 범위일 때, c 는 타일링 벡터이며,

c=(l,l+δ,l+2δ,…,u−2δ,u−δ)

δ 는 구간의 크기이고, u−l 는 δ 로 나누어 떨어지는 값입니다.

I+​(⋅) 가 입력값이 양수이면 1을 반환하고 그렇지 않으면 0을 반환하는 지시 함수일 때, 타일링 활성화 함수는 다음과 같습니다.

ϕ(z)=1−I+​(max(c−z,0)+max(z−δ−c))

타일링 활성화 함수는 강경한 경계를 갖고 있기 때문에 0 기울기를 갖는다는 것을 주의하세요.

퍼지 타일링 활성화

퍼지 지시 함수는 다음과 같습니다.

Iη,+​(x)=I+​(η−x)x+I+​(x−η)

퍼지 지시 함수는 0≤x<η 일 때, 0 부터 1 로 선형적으로 증가하고 η≤x 에서는 1 과 같아집니다. η 는 하이퍼 파라미터 입니다..

FTA 는 구간들 사이에 부드러운 경계를 만들기 위해 다음을 사용합니다.

ϕη​(z)=1−Iη,+​(max(c−z,0)+max(z−δ−c,0))
