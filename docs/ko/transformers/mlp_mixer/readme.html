#

MLP-Mixer: An all-MLP Architecture for Vision

이것은 MLP-Mixer: An all-MLP Architecture for Vision논문의 PyTorch 구현입니다..

이 논문은 비전 작업에 모델을 적용합니다. 이 모델은 패치(또는 NLP 작업의 경우 토큰)에 걸쳐 적용되는 MLP로 대체되는 어텐션 층이 있는 트랜스포머와 유사합니다.

MLP 믹서 구현은 트랜스포머 구현의 self-attention 레이어를 대체하는 것입니다. 따라서 시퀀스 차원에 걸쳐 MLP를 적용하기 위해 텐서를 전치하는 코드 몇 줄이면 됩니다.

논문에서는 비전 작업에 MLP Mixer를 적용했지만, 마스크된 언어 모델에서 이를 시도했습니다. 여기 실험 코드가 있습니다.
