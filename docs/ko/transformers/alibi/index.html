#

선형 편향이 존재하는 어텐션(ALiBi)

다음은 Train Short, Test Long: Attention with Linear Biases Enables Input Length Extrapolation 논문에서 제안한 선형 편향이 존재하는 어텐션(ALiBi)을 구현한 것입니다.

이것은 어텐션 점수(소프트 맥스 함수를 적용하기 전의 어텐션 로짓)에 더해진 편향이 있는 위치 인코딩을 대체합니다. 이것은 자기회귀 문제에서 테스트되는 상대적인 스키마이며 가까이있는 토큰들에서는 높은 편향성을, 멀리 떨어져있는 토큰들에서는 낮은 편향성을 가집니다. 편향성은 (소프트맥스를 적용하기 전이기 때문에) 로그 스케일로 점차 선형적으로 감소하며 각각의 헤드는 다른 기울기를 가집니다.

다음은 i번째 토큰에 대한 어텐션 공식입니다,

ai​​=softmax(qi​K⊤+m⋅[−(i−1),…,1,0])=softmax(qi​K⊤+m⋅[0,1,…,(i−1)])​

qi​∈Rd는 i번째 토큰에 대한 쿼리이며, K∈Ri×d는 i까지의 키입니다. 그리고 d는 각 헤드 당 특성의 개수입니다. softmax 가 평행 이동에 대해서 불변하기 때문에(모든 원소에 어떤 상수를 더해도 결과가 변하지 않습니다.) 위 등식은 항상 성립하지는 않습니다.

다음은 ALiBi 모델을 위한 훈련코드입니다.
