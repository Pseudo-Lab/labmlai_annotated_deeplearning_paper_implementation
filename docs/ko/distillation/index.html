신경망 지식 증류

이 문서는 신경망 지식 증류의 파이토치 구현 및 튜토리얼입니다.

이 기술은 대규모 네트워크의 표현력을 사용하여 작은 네트워크를 훈련하는 방법입니다. 즉 대규모 네트워크에서 지식을 추출하는 것입니다.

정규화되거나 드롭아웃을 통해 앙상블 효과를 학습한 큰 모델은 데이터와 레이블을 직접 학습할 때, 작은 모델보다 더욱 잘 일반화합니다. 이런 효과로 작은 모델은 큰 모델의 도움을 받아 성능을 높이거나 잘 일반화하도록 훈련이 가능합니다. 이렇게 학습된 작은 모델은 더 빠르고, 제한된 컴퓨팅과 메모리 등 실제 프로덕트 환경에서 유리합니다.

학습된 큰 모델의 출력 분포는 잘못된 클래스에도 0이 아닌 확률을 할당하기 때문에 실제 레이블보다 더 많은 정보를 제공합니다. 이러한 확률은 샘플이 특정 클래스마다 확률이 반드시 있음을 알려줍니다. 예를 들어, 숫자를 분류할 때 숫자 7이 주어지면 잘 학습된 모델은 7에는 높은 확률을, 2에는 0은 아니지만 의미있는 확률을 부여하고 다른 숫자에는 거의 0에 가까운 확률을 할당합니다. 지식 증류는 이 정보를 사용하여 강건하게 학습할 수 있습니다.

소프트 타겟

확률은 보통 소프트맥스 함수를 통헤 계산됩니다.

qi​=∑j​exp(zj​)exp(zi​)​

이때 qi​는 클래스의 확률이고, i and zi​는 로짓입니다.

KD의 목표는 보통 작은 모델의 출력 확률 분포와 큰 네트워크의 출력 확률 분포 사이의 교차 엔트로피 또는 KL 다이버전스를 최소화하는 것입니다.

주요한 문제 중 하나는 대규모 네트워크가 잘못된 클래스에 할당하는 확률이 매우 작아서 로스 계산에 기여하지 않는 경우가 많다는 것입니다. 따라서 온도 T를 적용하여 확률 분포를 부드럽게 제어할 수 있습니다.

qi​=∑j​exp(Tzj​​)exp(Tzi​​)​

당연하지만 T가 클수록 확률 분포는 더욱 부드러워집니다.

훈련

논문에서는 소규모 모델을 훈련할 때 실제 레이블을 예측하기 위한 두 번째 로스를 추가할 것을 제안합니다. 최종적으로 소프트 타겟과 실제 레이블이라는 두 가지 로스 항의 가중치 합으로 계산합니다.

증류용 데이터 세트를 전이 세트라고 하며, 논문에서는 동일한 훈련 데이터를 사용할 것을 제안합니다.

본 문서의 실험

CIFAR-10 데이터셋으로 학습합니다. 드롭아웃이 있는 14,728,266개의 파라미터가 있는 대규모 모델을 훈련한 결과, 유효성 검사 세트에서 85%의 정확도를 보였습니다. 437,034개의 파라미터가 있는 작은 모델은 80%의 정확도를 제공합니다.

그런 다음 큰 모델에서 증류하여 작은 모델을 훈련하면 정확도가 2% 증가하여 82%로 향상됩니다.

소프트 타겟입니다. pi​=∑j​exp(Tvj​​)exp(Tvi​​)​

#

141        soft_prob = nn.functional.log_softmax(output / self.temperature, dim=-1)
#

소프트 타겟 로스를 계산합니다. 

144        soft_targets_loss = self.kl_div_loss(soft_prob, soft_targets)
#

실제 레이블 로스를 계산합니다. 

146        label_loss = self.loss_func(output, target)
#

두 개의 로스를 가중합 합니다. 

148        loss = self.soft_targets_weight * soft_targets_loss + self.label_loss_weight * label_loss
#

로스를 기록합니다. 

150        tracker.add({"loss.kl_div.": soft_targets_loss,
151                     "loss.nll": label_loss,
152                     "loss.": loss})

155        self.accuracy(output, target)
156        self.accuracy.track()
