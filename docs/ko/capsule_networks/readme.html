#

캡슐 네트워크

다음은 캡슐간의 동적 라우팅을 파이토치로 구현하는 튜토리얼입니다.

캡슐 네트워크는 특성들을 캡슐로 임베드하고 투표 메커니즘을 통해서 다음 캡슐 레이어로 보내는 신경망 구조입니다.

단지 모듈로만 이번 컨셉을 이해하기는 어렵기 때문에, 이번 구현에서는 다른 모델들을 구현할 때와는 다르게 샘플을 포함하였습니다. 다음은 캡슐을 이용하여 MNIST 데이터를 분류하는 모델에 구현한 것으로 주석을 포함한 코드입니다.

이 파일은 캡슐 네트워크의 주요 모듈들에 대한 구현을 포함하고 있으며,

jindongwang/Pytorch-CapsuleNet을 참고하여 혼동되는 개념들을 정리하였습니다.
