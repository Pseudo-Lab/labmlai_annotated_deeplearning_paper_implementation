#

GPT-NeoX

이것은 Eleuther GPT-NeoX의 간단한 추론 및 파인튜닝 구현입니다.

  • 모델 정의
  • 토크나이저
  • 체크포인트 다운로드 및 로딩 헬퍼
  • 유틸리티
  • LLM.int8() 양자화

샘플

  • 텍스트 생성
  • 파이프라인 병렬화를 사용한 편향 미세 조정
  • LLM.int8()를 통한 텍스트 생성

평가

  • 단일 GPU에서 반 정밀도 모델 평가
  • LLM.int8() 모델 평가

공식 Eleuther GPT-NoeX 소스코드는 eleutherai/gpt-neox에 있습니다.
