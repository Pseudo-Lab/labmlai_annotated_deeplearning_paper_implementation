#

가중치 표준화

(Weight Standardization)

해당 내용은 논문 Micro-Batch Training with Batch-Channel Normalization and Weight Standardization에서의 가중치 표준화(Weight Standardization)에 대한 PyTorch 구현 입니다. 또한 Batch-Channel Normalization에 대한 구현도 포함되어있습니다.

배치 정규화(Batch normalization)는 완만한 손실의 형태를 제공하고 제거 특이점(Elimination singularities)을 배제합니다. 여기서 제거 특이점은 무용해진 네트워크의 노드입니다 (예: 항상 0을 출력하는 ReLU).

그러나 배치 정규화는 크기가 너무 작을 때 잘 작동하지 않으며, 이는 장치 메모리 한계로 인해 큰 네트워크를 훈련할 때 발생합니다. 해당 논문은 가중치 표준화와 Batch-Channel Normalization를 더 나은 대안으로 소개합니다.

가중치 표준화: 1. 기울기를 정규화한다 2. 손실 함수의 변동폭 감소한다 (Lipschitz constant 감소) 3. 제거 특이점 회피한다

Lipschitz 상수는 두 점 사이에서 함수가 가질 수 있는 최대 기울기입니다. 즉, L 은 Lipschitz 상수이며 L은 다음을 만족하는 최소값입니다. , ∀a,b∈A:∥f(a)−f(b)∥≤L∥a−b∥ where f:A→Rm,A∈Rn.

제거 특이점은 출력의 통계가 입력과 유사하게 유지되기 때문에 피할 수 있습니다. 따라서 입력이 정규 분포되어 있는 한, 출력은 정규에 가깝게 유지됩니다. 이는 노드의 출력이 활성화 함수의 활성 범위를 항상 벗어나지 않도록 합니다 (예: ReLU에 항상 음수 입력).

자세한 증명은 이 논문에서 찾을 수 있습니다.

여기에 CIFAR-10 데이터를 분류하기 위해 가중치 표준화를 사용한 VGG 네트워크 훈련에 대한 코드가 있습니다. 해당 코드는 가중치 표준화가 적용된 2D-Convolution Layer를 사용합니다.
