#

인스턴스 정규화

(Instance Normalization)

해당 내용은 논문 Instance Normalization: The Missing Ingredient for Fast Stylization에 대해 PyTorch로 구현해 본 내용입니다.

인스턴스 정규화(Instance normalization)는 style transfer를 개선하기 위해 도입되었습니다. 이는 스타일화(stylization)가 콘텐츠 이미지의 대비에 의존하지 않아야 한다는 결과에 기반합니다. "대비 정규화(contrast normalization)" 란

yt,i,j,k​=∑l=1H​∑m=1W​xt,i,l,m​xt,i,j,k​​

여기서 x는 t의 이미지의 인덱스, i는 feature 채널, 그리고 공간의 위치를 가진 이미지 배치 j,k입니다.

Convolutional network가 "대비 정규화"를 학습하기 어렵기 때문에, 해당 논문은 인스턴스 정규화를 도입하였습니다.

여기 CIFAR 10 분류 모델이 있으며, 이 모델은 인스턴스 정규화를 사용합니다.

인스턴스 정규화 레이어

인스턴스 정규화 레이어 IN은 입력 X 를 다음과 같이 정규화합니다.

입력 X∈RB×C×H×W 가 이미지 표현의 배치일 때, 여기서 B 는 배치의 크기, C 는 채널의 수, H 는 높이 그리고 W 너비 입니다. [γ∈RC 및 β∈RC]. 감마(gamma) 와 베타(beta)로 이루어진 아핀 변환(The affine transformation)은 선택 사항입니다.

IN(X)=γH,WVar​[X]+ϵ​X−H,WE​[X]​+β 

35class InstanceNorm(Module):
#
  • channels 는 입력 feature의 수 입니다.
  • eps 는 ϵ이며, Var[X]+ϵ​ 에서 수치 안정성을 위해 사용됩니다.
  • affine 은 정규화된 값을 조절하고 이동시키는지에 대한 여부입니다.

65        if self.affine:
66            self.scale = nn.Parameter(torch.ones(channels))
67            self.shift = nn.Parameter(torch.zeros(channels))

x 는 [batch_size, channels, *] 형태의 텐서(tensor)입니다. * 는 (0개일 수도 있는) 임의의 수의 차원을 나타냅니다. 예를 들어, 이미지(2D) 컨볼루션에서 해당 부분은 [batch_size, channels, height, width] 가 됩니다.

원래의 형태를 유지합니다. 

77        x_shape = x.shape

배치의 크기를 얻습니다. 

79        batch_size = x_shape[0]
#

feature 수가 동일한지 확인하기 위해 점검합니다. 

81        assert self.channels == x.shape[1]

[batch_size, channels, n] 으로 재구성합니다. 

84        x = x.view(batch_size, self.channels, -1)
#

마지막 차원을 따라 평균을 계산합니다. 즉, 각 feature의 평균 E[xt,i​]를 계산 

88        mean = x.mean(dim=[-1], keepdim=True)
#

첫 번째 및 마지막 차원을 따라 제곱 평균 계산합니다. 즉, 각 feature의 평균 E[(xt,i2​]를 계산. 

91        mean_x2 = (x ** 2).mean(dim=[-1], keepdim=True)
#

각 feature의 분산 Var[xt,i​]=E[xt,i2​]−E[xt,i​]2를 계산합니다. 

93        var = mean_x2 - mean ** 2
#

각 feature를 정규화 합니다.x^t,i​=Var[xt,i​]+ϵ​xt,i​−E[xt,i​]​ 

96        x_norm = (x - mean) / torch.sqrt(var + self.eps)
97        x_norm = x_norm.view(batch_size, self.channels, -1)
#

값을 스케일 및 이동시킵니다. yt,i​=γi​x^t,i​+βi​
