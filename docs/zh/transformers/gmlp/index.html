#

注意 MLP (GmLP)

这是 P yTorch 对《注意 MLP》一文的实现。

本文介绍了一种基于多层感知器（MLP）的带有门控的架构，他们将其命名为 gmLP。它由一堆L gmLP 块组成。

这是基于 GmLP 模型的自回归模型的训练代码。

gmLP Block

每个模块对输入嵌入进行以下转换，X∈Rn×d其中n是序列长度，d是嵌入的维度：

ZZ~Y​=σ(XU)=s(Z)=Z~V​

其中V和U是可学习的投影权重。s(⋅)是下面定义的空间门控单元。的输出维度s(⋅)将为的一半Z。σ是一个激活函数，比如 GeLU。

空间门控单元

s(Z)=Z1​⊙fW,b​(Z2​)

其中，fW,b​(Z)=WZ+b是沿序列维度的线性变换，⊙是逐元素乘法。Z被分成两个大小相等的部分，Z1​并Z2​沿着通道尺寸（嵌入维度）。
