长短期记忆细胞

LSTM 细胞计算c、和h。c就像长期记忆，h就像短期记忆。我们使用输入x和h来更新长期内存。在更新中，的c某些功能使用忘记门清除f，而某些功能i则通过大门添加g。

新的短期存储器是tanh长期存储器乘以输出门的值o。

请注意，在进行更新c时，单元格不会查看长期内存。它只会修改它。也c永远不会经历线性变换。这就是解决渐变消失和爆炸的问题。

以下是更新规则。

ct​ht​​=σ(ft​)⊙ct−1​+σ(it​)⊙tanh(gt​)=σ(ot​)⊙tanh(ct​)​

⊙代表逐元素乘法。

中间值和门计算为隐藏状态和输入的线性变换。

it​ft​gt​ot​​=linxi​(xt​)+linhi​(ht−1​)=linxf​(xt​)+linhf​(ht−1​)=linxg​(xt​)+linhg​(ht−1​)=linxo​(xt​)+linho​(ht−1​)​
