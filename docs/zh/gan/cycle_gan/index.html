Cyc@@

le GAN 进行图像到图像的转换。它训练模型将图像从给定分布转换到另一个分布，比如A类和B类的图像，某个分布的图像可以是某种风格或自然的图像。模型不需要 A 和 B 之间的配对图像，每个类别的一组图像就足够了。这非常适合在图像风格、光照变化、图案变化等之间进行切换。例如，将夏天改为冬天，将绘画风格改为照片，将马改为斑马。

Cycle GAN 可训练两个发电机模型和两个鉴别器模型。一个生成器将图像从 A 转换到 B，另一个从 B 转换到 A。判别器测试生成的图像是否真实。

重播缓冲区

重播缓冲区用于训练鉴别器。生成的图像被添加到重放缓冲区并从中取样。

重放缓冲区返回新添加的图像，概率为0.5。否则，它会发送一个较旧的生成的图像，并用新生成的图像替换旧的图像。

这样做是为了减少模型振荡。

288    def push_and_pop(self, data: torch.Tensor):
# 
290        data = data.detach()
291        res = []
292        for element in data:
293            if len(self.data) < self.max_size:
294                self.data.append(element)
295                res.append(element)
296            else:
297                if random.uniform(0, 1) > 0.5:
298                    i = random.randint(0, self.max_size - 1)
299                    res.append(self.data[i].clone())
300                    self.data[i] = element
301                else:
302                    res.append(element)
303        return torch.stack(res)

我们的目标是解决：G∗,F∗=argG,Fmin​DX​,DY​max​L(G,F,DX​,DY​)

其中，G翻译图像X→Y，F翻译来自的图像Y→X，DX​测试图像是否来自X太空，DY​测试图像是否来自Y太空，以及

L(G,F,DX​,DY​)LGAN​(G,F,DY​,X,Y)Lcyc​(G,F)Lidentity​(G,F)​=LGAN​(G,DY​,X,Y)+LGAN​(F,DX​,Y,X)+λ1​Lcyc​(G,F)+λ2​Lidentity​(G,F)=Ey∼pdata​(y)​[logDY​(y)]+Ex∼pdata​(x)​[log(1−DY​(G(x)))]+Ex∼pdata​(x)​[logDX​(x)]+Ey∼pdata​(y)​[log(1−DX​(F(y)))]=Ex∼pdata​(x)​[∥F(G(x))−x∥1​]+Ey∼pdata​(y)​[∥G(F(y))−y∥1​]=Ex∼pdata​(x)​[∥F(x)−x∥1​]+Ey∼pdata​(y)​[∥G(y)−y∥1​]​

LGAN​是原始 GAN 论文产生的对抗损失。

Lcyc​是循环损失，我们F(G(x))试图x与之相似和G(F(y))相似y。基本上，如果两个生成器（变换）是串联应用的，它应该返回原始图像。这是本文的主要贡献。它训练生成器以生成与原始图像相似的其他分布的图像。如果没有这种损失，G(x)可能会产生任何来自分发的损失Y。现在它需要从的分布中生成一些东西，Y但仍然具有的属性x，这样F(G(x)才能重新生成类似的东西x。

Lcyc​是身份丢失。这被用来鼓励映射以保留输入和输出之间的颜色构成。

为了求解G∗,F∗，鉴别DX​器和DY​应该在梯度上升，

∇θDX​,DY​​​m1​i=1∑m​​[logDY​(y(i))+log(1−DY​(G(x(i))))+logDX​(x(i))+log(1−DX​(F(y(i))))]​

这取决于负对数似然损失。

为了稳定训练，负对数似然目标被最小二乘损失所取代，即鉴别器的最小二乘误差，用1标记真实图像，将生成的图像标记为0。所以我们想在渐变上下降，

∇θDX​,DY​​​m1​i=1∑m​​[(DY​(y(i))−1)2+DY​(G(x(i)))2+(DX​(x(i))−1)2+DX​(F(y(i)))2]​

我们也使用最小二乘作为生成器。发电机应该下降到梯度上，

∇θF,G​​m1​i=1∑m​​[(DY​(G(x(i)))−1)2+(DX​(F(y(i)))−1)2+Lcyc​(G,F)+Lidentity​(G,F)]​

我们使用 fgenerator_xy orG 和 fgenerator_yx orF。我们使用 fdiscriminator_x orDX​ 和 fdiscriminator_y orDY​。
