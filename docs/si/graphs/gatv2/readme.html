#

ප්රස්තාරය අවධානය ජාල v2 (GATV2)

මෙයGATV2 ක්රියාකරුගේ PyTorch ක්රියාත්මක කිරීමකි ප්රස්ථාර අවධානය යොමු කරන ජාලයන් කෙතරම් අවධානයෙන් සිටිනවාද? .

GATV2sප්රස්තාර දත්ත මත ක්රියා කරයි. ප්රස්ථාරයක් නෝඩ් සහ දාර සම්බන්ධ කරන නෝඩ් වලින් සමන්විත වේ. උදාහරණයක් ලෙස, කෝරා දත්ත කට්ටලයේ නෝඩ් පර්යේෂණ පත්රිකා වන අතර දාර යනු පත්රිකා සම්බන්ධ කරන උපුටා දැක්වීම් වේ.

GATV2ක්රියාකරු සම්මත GAT හි ස්ථිතික අවධානය යොමු කිරීමේ ගැටළුව නිවැරදි කරයි: සම්මත GAT හි රේඛීය ස්ථර එකිනෙකට පසුව යොදන බැවින්, සහභාගී වූ නෝඩ් වල ශ්රේණිගත කිරීම විමසුම් නෝඩය මත කොන්දේසි විරහිතව පවතී. ඊට වෙනස්ව, GATV2 හි, සෑම නෝඩයකටම වෙනත් ඕනෑම නෝඩයකට සහභාගී විය හැකිය.

කෝරා දත්ත කට්ටලයේ ස්ථර දෙකක GATV2 පුහුණු කිරීම සඳහා පුහුණු කේතය මෙන්න.
